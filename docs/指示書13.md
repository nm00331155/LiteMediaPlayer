## 修正3: GestureOverlay.kt — シーク量計算改善

**ファイル**: `app/src/main/java/com/example/litemediaplayer/player/GestureOverlay.kt`

**変更箇所**: `calculateSeekDeltaMs` 関数を差し替え

**現在のコード**:
```kotlin
private fun calculateSeekDeltaMs(
    distanceX: Float,
    containerWidth: Float,
    seekIntervalSeconds: Int
): Long {
    if (containerWidth <= 0f) {
        return 0L
    }

    val ratio = (distanceX / containerWidth).coerceIn(-1f, 1f)
    val maxSeekMs = seekIntervalSeconds * 1000L
    return (ratio * maxSeekMs.toFloat()).roundToLong()
}
```

**変更後**:
```kotlin
private fun calculateSeekDeltaMs(
    distanceX: Float,
    containerWidth: Float,
    seekIntervalSeconds: Int
): Long {
    if (containerWidth <= 0f) return 0L

    // 画面幅の半分ドラッグで seekIntervalSeconds 秒分シーク
    // 画面幅全体ドラッグで最大 seekIntervalSeconds * 2 秒分シーク
    val halfWidth = containerWidth / 2f
    val ratio = (distanceX / halfWidth).coerceIn(-4f, 4f)
    val maxSeekMs = seekIntervalSeconds * 1000L
    return (ratio * maxSeekMs.toFloat()).roundToLong()
}
```

**補足**: 以前は `ratio` を `-1f..1f` にクランプしていたため、画面幅いっぱいドラッグしても `seekIntervalSeconds` 秒しかシークできなかった。変更後は半分ドラッグで同等、全体ドラッグで2倍、さらに超えて最大4倍までシーク可能。

            _uiState.update {
                it.copy(errorMessage = "先に設定タブでグローバルロックを設定してください")
            }
            return@launch
        }

        lockConfigDao.upsert(
            com.example.litemediaplayer.data.LockConfig(
                targetType = LockTargetType.COMIC_FOLDER.name,
                targetId = folderId,
                authMethod = global.authMethod,
                pinHash = global.pinHash,
                patternHash = global.patternHash,
                autoLockMinutes = global.autoLockMinutes,
                isHidden = global.isHidden,
                isEnabled = true
            )
        )
    }
