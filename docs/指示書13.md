問題 2: シーク時間が1秒単位のまま（設定無視）
原因: GestureOverlay.kt の calculateSeekDeltaMs は正しく seekIntervalSeconds を使用していますが、ドラッグの幅に対する比率が (-1, 1) に制限されているため、画面端まで指を移動してもシーク量が最大 seekIntervalSeconds 秒にしかなりません。短いドラッグだと1秒前後にしかならないので「1秒固定」に見えます。

修正ファイル: GestureOverlay.kt

修正内容: calculateSeekDeltaMs を以下に置換

Copyprivate fun calculateSeekDeltaMs(
    distanceX: Float,
    containerWidth: Float,
    seekIntervalSeconds: Int
): Long {
    if (containerWidth <= 0f) return 0L
    
    // 画面幅の半分ドラッグで seekIntervalSeconds 秒分のシーク
    // これにより短い指移動でも意味のあるシーク量が得られる
    val ratio = distanceX / (containerWidth * 0.5f)
    val maxSeekMs = seekIntervalSeconds * 1000L
    return (ratio * maxSeekMs.toFloat()).roundToLong()
        .coerceIn(-maxSeekMs * 4, maxSeekMs * 4) // 最大4倍まで
}
